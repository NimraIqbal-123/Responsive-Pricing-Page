# Responsive Pricing Page

## Overview

A modern and fully responsive pricing page built using HTML5, CSS3, and Bootstrap 5. The project features a clean navigation bar, responsive pricing cards, call-to-action buttons, and a professional footer. It is optimized for desktop, tablet, and mobile devices, demonstrating responsive web design principles and effective use of Bootstrap components.

## Features

* Responsive Navigation Bar
* Modern Pricing Cards
* Mobile-Friendly Layout
* Bootstrap 5 Grid System
* Call-to-Action Buttons
* Hover Effects and Smooth Transitions
* Professional Footer
* Clean and Minimal UI Design
* Cross-Browser Compatibility

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Remix Icons
* JavaScript

## Purpose

This project showcases frontend development skills by creating a professional pricing page commonly used in SaaS, startup, and business websites. It demonstrates responsive layouts, Bootstrap components, modern UI design, and clean project organization suitable for real-world web development and portfolio presentation.
